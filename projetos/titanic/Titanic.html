Titanic: quem morreu e quem sobreviveu?¶

O Titanic foi um navio construdo na Irlanda, que naufragou quatro dias aps sua viagem de inaugurao em abril de 1912. Essa acidente muito conhecido, ainda mais depois do lanamento do filme em 1997. Esse projeto visa criar um modelo a fim de predizer quem morreu e quem sobreviveu nesse desastre, com base no Dataset do Titanic do Kaggle.

Utilizaremos uma verso semi-limpa do dataset e o modelo de Regresso Logstica para classificar os que morreram e os que sobreviveram.

Esse conjunto de dados contém as seguintes informações:¶

  • PassengerId: nmero de identificao dos passageiros
  • Survived: informao se sobreviveu o morreu (0=no e 1=sim)
  • Pclass: classe ocupada (1, 2 ou 3)
  • Name: nome do passageiro
  • Sex: sexo
  • Age: Idade
  • SibSp: acompanhado de irmos ou esposo(a)s
  • Parch: acompanhado dos pais / crianas a bordo
  • Ticket: nmero da passagem
  • Fare: valor pago pela passagem
  • Cabin: nmero da cabine
  • Embarked: porto de embarcao (C = Cherbourg, Q = Queenstown, S = Southampton)

Limpando os dados¶

Agora que j conhecemos melhor nossos dados, precisamos tratar os dados faltantes. Comecemos pela idade, se simplesmente excluirmos essas dados, perderemos muita informao importante. Por isso, uma forma de solucionarmos o problema fazer uma mdia de todos os passageiros de acordo com sua classe e preencher os dados faltantes.

In [70]:
def add_age(cols):
    age = cols[0]
    pclass = cols[1]
    if pd.isnull(age):
        if pclass == 1:
            return 37
        elif pclass == 2:
            return 29
        elif pclass == 3:
            return 24
    else:
        return age
